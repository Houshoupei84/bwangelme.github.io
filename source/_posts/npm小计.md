## npm ls

### 用法

```
    npm ls [[<@scope>]<pkg> ...]
    别名: list, la, ll
```

### 说明

+ 这条命令将会在stdout中打印出所有已经安装的包和他们的依赖包，以树形目录的形式打印出来。

+ 位置参数是`name@version-range`形式的标识符，它将会打印结果，只打印出命名包的路径中所含有的包。

+ 可以通过加上-g参数来打印全局安装的包。
